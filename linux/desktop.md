# desktops for ubuntu

* GNOME 3

  default to 18.04.

  ```bash
  sudo tasksel install ubuntu-desktop
  ```

  

* KDE Plasma

  ```bash
  sudo tasksel install kubuntu-desktop
  ```

  

* Mate

  ```bash
  sudo tasksel install ubuntu-mate-core
  ```

  

* Xubuntu

  ```bash
  sudo tasksel install xubuntu-core
  ```
