# 动态规划

### 前提

* 最优子结构：问题的解可以通过组合子问题的解得到，即可以通过状态转移方程表示。 

* 重叠子问题：求解过程中子问题重叠。

* 通常是求最值问题。

### Knapsack

##### 01

状态方程：$f[i][j]$表示使用前$i$个物品，最大容量为$j$时可以获得的最大价值。

$$
f[i][j] = \max(f[i-1][j], f[i-1][j-w[j]]+v[j])
$$

边界条件：

* $f[0][j] = 0$
* $f[i][0] = 0$****

```c++
const int M = 100 + 1;
const int N = 1000 + 1;

int ws[M];
int vs[M];

int dp[M][N];

int main() {
    // inputs
    int n, m;
    cin >> n >> m;
    for (int i = 0; i < m; i++) {
        cin >> ws[i] >> vs[i];
    }
    // init
    memset(dp, 0, sizeof(dp));
    // loop
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            dp[i][j] = dp[i-1][j];
            if (j > ws[i]) dp[i][j] = max(dp[i][j], dp[i-1][j-ws[i]]+vs[i]);
        }
    }
    // return
    cout << dp[m][n] << endl;
    return 0;
}
```

空间优化：

```c++
const int M = 100 + 1;
const int N = 1000 + 1;

int ws[M];
int vs[M];

int dp[N];

int main() {
    // inputs
    int n, m;
    cin >> n >> m;
    for (int i = 0; i < m; i++) {
        cin >> ws[i] >> vs[i];
    }
    // init
    memset(dp, 0, sizeof(dp));
    // loop
    for (int i = 1; i <= m; i++) {
        // reversed order! dp[i][j] only use dp[i-1][k] where k < j
        for (int j = n; j >= 1; j--) {
            if (j > ws[i]) dp[j] = max(dp[j], dp[j-ws[i]]+vs[i]);
        }
    }
    // return
    cout << dp[n] << endl;
    return 0;
}
```

##### Complete

$$
f[i][j] = \max_{0 \le k \le K}(f[i-1][j-kw[i]] + kv[i]) \\
f[i][j] = \max(f[i-1][j], f[i][j-w[i]]+v[i])
$$

### 题目

##### [完全平方数](https://leetcode-cn.com/problems/perfect-squares/)

```c++
class Solution {
public:
    int numSquares(int n) {
        // complete knapsack
        int m = sqrt(n) + 1;
        // init
        vector<int> dp(n + 1, 0x3f3f3f3f); // dp[1~n] should be inf
        dp[0] = 0; // but dp[0] is 0
        // loop
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (j - i * i >= 0) dp[j] = min(dp[j], dp[j - i * i] + 1);
            }
        }
        return dp[n];
    }
};
```
